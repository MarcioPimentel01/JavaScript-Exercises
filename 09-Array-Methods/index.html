Why use an array?

If you have a list of items (a list of car names, for example), storing the cars in single variables could look like this:

                let car1 = `saab`
                let car2 = `volvo`
                let car3 = `volkswagen`
                let car4 = `BMW`
                let car5 = `Ford`

However, if you want to loop through the cars and find a specifc one? Imagine also that instead of five cars you have 200! A good solution is and array, because can hold a large amount of values under a single name. And you can access the values by refering the index number.

Creating an array

const array_name = [item1, item2, ...];

Using our example above, would it be:

const array_carBrands = [Saab, volvo, Volskwagen, BMW, Ford];

You can also create arrays Using the JavaScript Keyword "new"

When you use the keyword "new" you can create thae array and also asigns values to it.

See sintax bellow:

Accessing Array Elements

You access an array element by referring to the, Index number

This statement changes the value of the first "[0]" element in, FordModels
