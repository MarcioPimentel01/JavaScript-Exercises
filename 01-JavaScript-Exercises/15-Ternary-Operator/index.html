15 Ternary Operator

The ternary operator, often referred to as the conditional operator, is a concise way to write conditional expressions in JavaScript. Its syntax is:

        
            condition ? expr1 : expr2

Here's how it works:

  • First, the condition is evaluated.
  • If the condition is true, the expression evaluates to expr1.
  • If the condition is false, the expression evaluates to expr2.

It's commonly used when you want to assign a value to a variable based on a simple condition. For example:

            
                let result = (age >= 18) ? "Adult" : "Minor";

This assigns the string "Adult" to the variable result if age is greater than or equal to 18, otherwise, it assigns "Minor".

However, the ternary operator is limited to handling binary decisions - it can only choose between two outcomes. If you need to handle more than two outcomes or more complex conditions, it's often better to use if...else statements for clarity and readability.
